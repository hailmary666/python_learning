##  Проект: Анализ динамики населения мира
**Цель:** Научиться работать с реальными данными в Pandas и создавать базовые визуализации.
*   **Источник данных:** Датасет `world_population.csv` (прилагается или ссылка на источник).
*   **Что было сделано:**
    *   Загрузка и первичный осмотр данных с помощью `df.head()`, `df.info()`.
    *   Визуализация: построены линейные графики изменения населения для отдельных стран, столбчатая диаграмма для топ-5 стран по населению за 2022 год.
    *   **Ключевой вывод:** Например, "Население мира продолжает расти, но темпы роста распределены неравномерно. Топ-5 стран (Китай, Индия, США, Индонезия, Пакистан) составляют более 40% от общего населения."
